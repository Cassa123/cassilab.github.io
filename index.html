01 / About
"I write code for a living and spend the rest of the time figuring out what I don't know yet."

My journey into software development didn't start with fancy frameworks or modern ecosystems. My first role at W3 Technologies had me building front-end applications using nothing but PHP, HTML, and CSS. Back then, Angular, Spring Boot, and cloud architecture were mysterious creatures I'd only heard whispers about.

Fast forward a few years, and I've transformed into an Intermediate Software Developer at Ideal Solution, where I dive headfirst into Angular, Java, Spring Boot, TypeScript, APIs, orchestration layers, AWS, and software architecture. Every day feels like solving puzzles — which suits me perfectly, because I genuinely enjoy figuring out how things work, and more importantly, how they might break.

I work carefully, ask the right questions before they become the wrong problems, and have a reasonable track record of catching things that probably shouldn't make it to production. I'm still learning — but deliberately, which I think counts for something.

02 / Work at Ideal Solution

A selection of projects I contributed to as a Software Developer at Ideal Solution. Each one taught me something new — from leaderboard systems to payment gateways to iOS exploration.

Kaizen

A Jira-integrated platform that gamified the Software Development Lifecycle (SDLC), transforming ticket completion into measurable achievements, rankings, and performance insights. The platform provided developers and teams with visibility into their contributions through competitive leaderboards.

Key Contributions:

  • Redesigned the frontend experience to improve usability and readability
  • Implemented weekly, monthly, and yearly leaderboards for both developers and teams, providing visibility into performance trends over time.
  • Built an automated scheduling and notification system that delivers weekly and monthly leaderboard standings and score summaries via email.
  • Integrated with Jira webhooks to capture ticket lifecycle events, automatically detecting when issues transitioned to a completed state and processing associated story points for leaderboard scoring and performance tracking.
Angular Java TypeScript Spring Boot Jira API Webhooks

Glimmer

Glimmer is Paycorp Group's digital merchant onboarding platform, delivering a mobile-first onboarding experience for POS acquiring merchants. Built to support both sole proprietors and complex company structures, the platform enables an onboarding journey with a target turnaround time of under 15 minutes.

Key Contributions:

  • Architected and delivered the complete, fully responsive, frontend onboarding experience using Syncfusion Angular component library.
  • Integrated the Sybrin SDK to support identity verification, screening, address validation, and AVS checks as part of the compliance process.
  • Developed a scalable white-label solution, enabling ISO partners to customize branding and merchant-facing experiences.
  • Implemented workflow controls and validation mechanisms to protect onboarding data integrity and reduce opportunities for fraudulent submissions.
  • Extended contributions beyond the frontend by supporting backend development, resolving bugs, and implementing new platform features.
  • Executed SQL update queries to prepare and modify data within testing environments.
Angular Java TypeScript Spring Boot SDK Syncfusion

Payments Hub

A payment processing hub that centralizes merchant settlements, automates payment routing, and enables flexible rule-based transaction handling.

Key Contributions:

  • Architected and delivered the base structure of the Payments Hub portal using Syncfusion Angular component library.
  • Integrated Microsoft Active Directory (Azure AD) authentication, enabling secure single sign-on (SSO) access for internal users.
  • Extended contributions beyond the frontend by supporting backend development, resolving bugs, and implementing new platform features.
Angular Java TypeScript Spring Boot Microsoft AD Syncfusion

Coffee / Brewflow App

Brewflow is a platform that enables digital ordering, queue orchestration, and operational efficiency for beverage shops, starting with coffee retailers and designed for broader retail applications. Built for flexibility and scale, the system supports multi-tenant, multi-store deployments with integrated POS, mobile ordering, and staff workflows.

Key Contributions:

  • Developed the iOS POS application used by retail staff to view, manage, and fulfill incoming customer orders.
  • Collaborated on database design and architecture, defining entity relationships and data models to support order management, store operations, and multi-tenant functionality.
  • Contributed to the setup and configuration of AWS infrastructure, supporting the deployment and operation of the platform.
Java iOS Mobile AWS

Transaction Junction - Internet Payment Gateway

A leading FinTech company that provides a modern transaction platform to businesses across numerous industries.

Key Contributions:

  • Contributed to the implementation of Apple Pay as a new payment method, supporting the expansion of the platform's digital payment capabilities.
  • Resolved support tickets and investigated production issues, working closely with QA teams to diagnose and fix defects identified during testing.
  • Refactored database access layers by converting native SQL queries to Hibernate Query Language (HQL), improving maintainability and alignment with application standards.
  • Redeveloped the Merchant Portal frontend using React, modernizing the user interface and improving usability, responsiveness, and overall user experience.
